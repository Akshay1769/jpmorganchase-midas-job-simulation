##  jpmorganchase-midas-job-simulation

A Spring Boot-based backend system that simulates a financial transaction pipeline using Kafka messaging, user account management, and real-time balance updates.

🚀 Features

📥 Kafka Producer Integration
Sends transaction data to Kafka topics

👤 User Data Population
Loads user records from file into database

💸 Transaction Processing
Parses transactions and updates balances

🧹 Robust Input Handling
Handles malformed inputs, extra spaces, and multi-line data safely

⚡ Real-Time Updates
Processes and reflects balance changes instantly
